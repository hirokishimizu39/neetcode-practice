# NeetCode 150 Practice

Big Tech (Amazon, Google, Meta, etc.) 技術面接対策のための練習リポジトリ

## 目標
- 1日1問を習慣化
- C++, Python, TypeScript, Go の4言語でマスター
- 丸暗記レベルまで周回

---

## 習慣化のコツ

### [小さく始める](https://jamesclear.com/atomic-habits-summary)
- 最初は1日1問、15-30分から
- 「毎日解く」より「毎日コードを開く」を目標に
- 朝の固定時間がおすすめ（意志力が高い）

### [ロードマップに従う](https://neetcode.io/roadmap)
- カテゴリ順（Arrays → Two Pointers → ...）に進める
- Easy → Medium → Hard の順
- 同じパターンを連続で解くと定着しやすい

### [解けるようになるコツ](https://blog.algomaster.io/p/how-to-start-leetcode-in-2025)
1. **15-20分で解けなければ解答を見る** - 悩みすぎは時間の無駄
2. **解答を「理解」してから、翌日に再挑戦**
3. **パターン認識を重視** - 「これはスライディングウィンドウだ」と分かることが目標

### 復習サイクル
| Day | アクション |
|-----|-----------|
| Day 1 | 解く（解けなければ解答を見る） |
| Day 2 | 何も見ずに再挑戦 |
| Day 7 | もう一度解く |
| Day 30 | 最終確認 |

### 実践的なアドバイス
- **解説動画を活用**: NeetCodeのYouTubeは非常にわかりやすい（各問題のリンクは下表参照）
- **手書きで図を描く**: 特にLinked List、Tree、Graphは視覚化が重要
- **時間計測**: 本番を意識して制限時間を設ける

> 最初の1-2ヶ月は「解けない」のが普通。パターンが身につくと急に解けるようになる。
